# DOMANIA
## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/) (Node Package Manager)


### Installation

1. Clone the repo
   ```sh
   gh repo clone D1ANAND/DOMANIA
   
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
3. Start project
   ```sh
   npm start
   ```
